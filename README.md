# DQN_TicTacToe
Contains both the gym environment for the Tic-Tac-Toe, as well as a DQN implementation to train model.

## How to run

```
$ cd DQN_tictactoe/gym_tictac4
$ pip3 install -e .
$ python3 dqn_tictac.py
```
